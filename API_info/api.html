Монохромы
Параметры фильтрации (Query Params):
  • top=N — Лимит количества результатов
  • coof=0.X — Минимальный порог схожести от 0.0 до 1.0
Эти параметры можно комбинировать в любом запросе этой секции.
POST
MonoCoof/TopNftByColor

Поиск лучших моделей, подходящих под заданный цвет фона.

Request Body
{
  "NameGift": "Lush Bouquet",
  "NameColor": "Carrot Juice",
  "MonohromeModelsOnly": true
}
Response Example
[
    { 
        "Name": "Tulips", 
        "Coof": 0.93 
    },
    { 
        "Name": "Carrots", 
        "Coof": 0.90 
    }
]
POST
MonoCoof/TopBackgroundColorsByNFT

Подбор лучших фонов для конкретной модели.

Request Body
{
  "NameGift": "Lush Bouquet",
  "NameModel": "Carrots"
}
Response Example
[
    {
        "Key": "Persimmon",
        "Value": 0.92
    },
    {
        "Key": "Carrot Juice",
        "Value": 0.90
    },
]
POST
MonoCoof/TopBackgroundColorsByColors

Поиск фонов для списка цветов (HEX). Рекомендуется указывать 3 оттенка одного цвета, близких друг к другу.

Request Body
{
  "Colors": 
  [
    "#FF5500", 
    "#FF4500", 
    "#FF6347"
  ]
}
Response Example
[
    {
        "Key": "Tomato",
        "Value": 0.91
    },
    {
        "Key": "BurntSienna",
        "Value": 0.81
    }
]
POST
MonoCoof/MonohromeCoof

Расчет точного коэффициента сочетаемости (0.0 - 1.0) для конкретной пары.

Request Body
{
    "NameGift": "Lush Bouquet",
    "NameModel": "Carrots",
    "NameColor": "Carrot Juice"
}
Response Example
{
    "Coof": 0.90,
    "Ismonohrome": true
}

Тематики
Параметры списка (Query Params): Эти параметры применяются к методам GetAllCollections:
  • page=1 — Номер страницы (int)
  • search=text — Поиск по названию (string)
  • sort=name|count — Сортировка по имени или количеству
  • desc=true|false — Направление сортировки
GET
Thematic/GetAllCollections

Получение базового списка коллекций (ID, Название, Количество).

GET
Thematic/GetAllCollections/WithParameters

Получение расширенного списка коллекций (включая дополнительные данные).

GET
Thematic/GetGiftsByCollection/{Name}

Получить простой список моделей, входящих в коллекцию.

GET
Thematic/GetGiftsByCollection/{Name}/WithParameters/{Bg?}

Получить модели с кластеризацией. Параметр {Bg?} является опциональным (название фона).

GET
Thematic/GetCollectionByGift/{Gift}/{Model}

Узнать, к каким тематикам относится конкретная модель (простой список имен).

GET
Thematic/GetCollectionByGift/{Gift}/{Model}/WithParameters

Узнать тематики модели с расширенной информацией.
